# probot-security-vulnerabilities

> A GitHub App built with [Probot](https://github.com/probot/probot) that File issues for GitHub security vulnerabilities alerts

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
