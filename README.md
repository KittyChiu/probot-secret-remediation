# probot-secret-remediation

> A GitHub App built with [Probot](https://github.com/probot/probot) that creates remediation issue for push protection fix-later bypass.

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
